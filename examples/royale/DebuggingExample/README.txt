DESCRIPTION

The DebuggingExample shows how to use the debugging package.

The debugging functions will only be called in the debug build and will do nothing in the release build.

This Royale application may be run as a Flash SWF or cross-compiled (using the Royale Compiler)
into JavaScript and HTML and run without Flash.


COMPONENTS and BEADS

- TextButton
- Label
- TextArea


NOTES

The Changing the content of the TextArea will reverse the passing and failing asserts.
